# Bomberman Recreation

This project is a recreation of the classic game Bomberman.  
It showcases simple but iconic gameplay mechanics such as character movement, placing bombs that explode after a delay, and collecting power‑ups that change how the match plays out. 

## Objective

- A player wins when the other player dies.  
- Use power‑ups to gain an advantage and control the map.

## Features

- Walking in four directions  
- Sprite‑based animations for movement and bombs  
- Bombs that explode after a delay; bombs can be pushed during the windup period  
- Breakable walls that have a chance to drop items  
- Local 2‑player multiplayer  
- Simple, readable implementation of classic Bomberman mechanics

## Controls

**Player 1**  
- Movement: WASD  
- Drop Bomb: Space  

**Player 2**  
- Movement: Arrow Keys  
- Drop Bomb: Numpad 1
