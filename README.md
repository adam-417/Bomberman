# Bomberman Recreation 
This is a recreation of the classic game: Bomberman. 
The project demonstrates simple game mechanics like movement, 
placing bombs that explode and picking up power ups. 

Objective 
-A player wins when the other dies. 
-Use powerups to gain an advantage.

Features 
-Walking in 4 directions  
-Animation for walking and bombs using sprites.
-Placing bombs down that explode after a delay, bombs can be 
pushed during the windup time for the explosion.
-Breakable walls (have a chance to drop items).  
-Local 2 player multiplayer  

Controls 
-WASD for walking (arrow keys for player 2). 
-SPACE for dropping bombs (numpad1 for player 2).
